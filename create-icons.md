# Como criar os ícones da PWA

Os ícones devem ser criados manualmente ou usando uma ferramenta online.

## Tamanhos necessários:
- icon-192x192.png (192x192 pixels)
- icon-512x512.png (512x512 pixels)

## Sugestões de ferramentas:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

## Design sugerido:
- Fundo: #8B0000 (vermelho escuro)
- Ícone: Pokéball ou símbolo de Pokédex
- Estilo: Futurista, minimalista

Coloque os arquivos gerados na pasta `public/`
